# 🎨 生图Prompt提取API仓库

## 📖 项目简介

这是一个专门用于从GitHub issue中提取、整理和管理AI生图prompt的仓库。通过精简的YAML issue模板和零依赖的自动化系统，我们可以高效地收集和分享高质量的prompt、模型配置和生成效果图。

## 🚀 主要功能

- **精简Prompt提取**: 使用8个核心字段的YAML模板，快速提交prompt信息
- **零依赖自动化**: 基于官方GitHub CLI，无需Node.js或其他依赖
- **多模态模型支持**: 支持Stable Diffusion、Midjourney、DALL-E、Gemini等多种模型
- **NSFW内容标记**: 复选框方式标记内容安全性
- **智能图片处理**: 支持外部URL和GitHub Issue图片链接
- **标签分类系统**: 通过逗号分隔的标签便于搜索和管理
- **定时数据同步**: 每天自动提取已关闭issue数据
- **统一JSON输出**: 所有数据整合到一个JSON文件中

## 🎯 Issue模板核心字段

### 必填字段 (6个)
- **🎯 Prompt标题**: 简洁描述prompt的主要内容
- **👤 作者**: prompt的原创作者或发现者
- **📄 完整Prompt**: 完整的prompt文本
- **🤖 模型名称**: 使用的AI模型名称
- **🏷️ 标签**: 用逗号分隔的标签，如：赛博朋克,人物,科幻
- **🔞 NSFW内容**: 复选框选择安全内容或成人内容

### 可选字段 (2个)
- **⚙️ 其他参数**: 采样方法、步数、CFG Scale、温度等
- **🖼️ 效果图URL**: 图片链接或GitHub上传的图片

## 🏷️ 标签使用规范

### 标签示例
- **风格**: 赛博朋克、写实、动漫、油画、像素艺术
- **内容**: 人物、肖像、风景、动物、建筑、抽象
- **技术**: 高质量、4K、8K、专业摄影、电影级光照
- **用途**: 壁纸、插画、游戏素材、设计灵感
